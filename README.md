# SkilletHub

> A webapp for collaboratively developing recipes, inspired by github.

## Requirements

- node v6.4.0
- express v4.14.0

### Installing Dependencies

From within the root directory:

```sh
npm install
```

Transpile, concat, and minify with:
```sh
grunt build
```

## Usage

Start the server with
```sh
node server/server.js
```
